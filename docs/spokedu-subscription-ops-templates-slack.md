# 운영 템플릿 — 슬랙/채팅용 복붙

[`spokedu-subscription-ops-templates.md`](./spokedu-subscription-ops-templates.md) §2와 동일 문안을 줄바꿈만 슬랙에 맞게 정리했습니다.

---

## 템플릿 A: 자동 결제 문의

```
안녕하세요. 현재 자동 결제는 준비 중이며, 플랜 전환은 운영팀이 수동으로 안내드리고 있습니다.
원하시는 플랜(예: Basic/Pro)과 센터 정보를 보내주시면 바로 도와드리겠습니다.
```

---

## 템플릿 B: 체험 종료 안내

```
안녕하세요. 체험 기간이 종료되어 Free 플랜으로 전환되었습니다.
계속 이용을 원하시면 회신 주시면 전환 절차를 빠르게 안내드리겠습니다.
```

---

## 템플릿 C: 결제 지연 안내

```
안녕하세요. 현재 구독 결제가 확인되지 않아 결제 지연 상태로 전환되었습니다.
결제 확인 후 바로 활성 상태로 복구해 드리겠습니다.
```

---

## 템플릿 D: 결제 지연 복구 완료

```
안녕하세요. 결제 확인이 완료되어 활성 상태로 정상 복구되었습니다.
현재 기존 플랜 기능을 동일하게 이용하실 수 있습니다.
```

---

## 템플릿 E: 해지 접수

```
안녕하세요. 해지 요청이 정상 접수되었습니다.
적용 시점과 이후 이용 범위를 정리해 곧 안내드리겠습니다.
```
