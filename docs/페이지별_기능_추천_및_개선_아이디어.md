# Admin 페이지별 기능 추천 및 개선 아이디어

> **목적**: 각 Admin 페이지의 실제 사용 흐름과 운영痛点을 반영한 실용적 기능·개선 제안  
> **원칙**: 기존 구조·데이터 모델을 최대한 활용하고, 일상 업무 효율에 직결되는 항목 우선

---

## 1. 대시보드 (`/admin`)

### 1.1 현황
- 오늘 수업, 일정 요약, 업무 노트(공통+담당자별), 수업 연기 알림 표시
- New Task, Daily Note 모달, 새로고침

### 1.2 추천 기능

**오늘 수업 바로가기**
- 수업 행 클릭 시 수업 관리 캘린더의 해당 날짜로 이동
- “이 수업 편집” 클릭 한 번으로 `/admin/classes` 해당 세션 모달까지 연결

**업무 노트 작성 가속**
- 카드에서 “Daily Work Note” 없이도 메모 입력 가능한 인라인 필드
- 혹은 카드 헤더 옆 짧은 입력창으로 “오늘 한 줄” 빠르게 기록

**수업 연기 알림 활용도 향상**
- 휴가 등록/해제는 `users` 테이블 기반이므로, “휴가 등록” 버튼 → 강사 선택 모달로 `users.vacation` 수정
- “확인함” 처리로 연기 알림 숨기기(선택): `memos` 또는 별도 테이블에 “확인한 연기 ID” 저장

**일정·업무 한눈에**
- 일정 “상태” 컬럼 색상 강화(진행중/종료 등)
- 오늘·내일 일정 개수 배지, 또는 “D-day” 스타일 표시

---

## 2. 일정 (`/admin/schedules`)

### 2.1 현황
- 테이블/드로어 기반 CRUD, 검색·상태 필터, 인라인 편집

### 2.2 추천 기능

**반복 일정 생성**
- “이 일정을 N주 반복” 옵션으로 동일 패턴 여러 주 자동 생성
- 기간·요일 선택 후 일괄 INSERT

**일정 템플릿**
- 자주 쓰는 일정(예: 정기 점검, 교육)을 템플릿으로 저장
- 새 일정 생성 시 “템플릿에서 불러오기”로 제목·담당·체크리스트 등 복사

**담당자·기간별 요약**
- 상단에 “이번 달 일정 N건 / 진행중 M건” 등 요약 카드
- 담당자별 필터 + 해당 담당자 일정 건수 표시

---

## 3. 센터 관리 (`/admin/centers`)

### 3.1 현황
- 센터 목록, 상세(Operations/Programs/Finance/Logs/Files), CRUD

### 3.2 추천 기능

**계약 기간 경고**
- `contract_end` 기반 “D-30”, “D-7” 배지
- 계약 만료 임박 센터를 대시보드나 센터 목록 상단에 요약 표시

**Next Actions 사용성**
- 체크리스트 형태 유지하면서 “추가/삭제/순서 변경”을 모달 없이 인라인으로
- 완료 시 완료일 자동 기록(별도 컬럼 추가 시)

**프로그램·재무 연계**
- 센터 상세에서 “이번 달 프로그램 N건”, “미정산 항목 M건” 등 한 줄 요약
- 클릭 시 Programs/Finance 탭으로 포커스

---

## 4. 수업 관리 (`/admin/classes`)

### 4.1 현황
- FullCalendar, 세션 편집/연기/복제/삭제, 수동·자동 완료, session_count_logs 연동

### 4.2 추천 기능

**강사별 일일 요약**
- 캘린더 옆 또는 상단에 “오늘 강사별 수업 수” 요약
- 특정 강사 클릭 시 해당 강사 수업만 필터

**대량 일정 생성**
- “N주차, 주 M회, O요일” 등 규칙 기반 반복 수업 생성
- 기존 `handleCloneGroup` 로직을 확장해 “규칙 기반 복제” 모드 추가

**수업 로그 내보내기**
- 기간·강사 선택 후 `session_count_logs` + `sessions` 기준 CSV/Excel 내보내기
- 강사별 수업 수·세션 제목 등 포함

**연기/취소 시 시각적 구분**
- 캘린더에서 연기/취소 이벤트 색상·스타일을 더 명확히 구분
- 범례(legend)로 상태별 의미 안내

---

## 5. 공지사항 (`/admin/notice`)

### 5.1 현황
- 카테고리(필독/일반/이벤트), 고정, CRUD, 카드 형태 표시

### 5.2 추천 기능

**읽음 표시**
- 공지별 “읽음” 상태 저장(예: `notice_reads` 테이블 또는 사용자별 JSON)
- 읽지 않은 공지 수 배지, 필독 미읽음 강조

**예약 발행**
- `published_at` 컬럼 추가 후, 특정 시각에 자동 공개
- 예약 목록 탭에서 예약 공지 관리

**검색·필터**
- 제목·내용 검색
- 카테고리·기간·고정 여부 필터

---

## 6. 연간 커리큘럼 (`/admin/curriculum`)

### 6.1 현황
- 월/주차 선택, 커리큘럼 CRUD, 유튜브·인스타 링크, Expert Tip, 체크리스트, 교구, 활동 방법

### 6.2 추천 기능

**순서 변경**
- 동일 주차 내 커리큘럼 드래그로 순서 변경
- `sort_order` 또는 `display_order` 컬럼 추가 후 업데이트

**커리큘럼 복사**
- 기존 커리큘럼을 “다른 월/주차로 복사”
- 제목·URL·체크리스트 등 복사 후 수정

**검색**
- 제목·태그(별도 컬럼 추가 시) 검색
- 월/주차 필터와 조합

---

## 7. 교구/재고 (`/admin/inventory`)

### 7.1 현황
- 카탈로그, 강사별 재고, 드래그 배부, 수량 수정, 로그

### 7.2 추천 기능

**최소 재고 알림**
- 카탈로그에 `min_quantity` 추가
- 강사별 재고가 최소치 이하일 때 배지·경고 표시

**교구별 이력 요약**
- 특정 교구 클릭 시 “최근 N일 입·출고 이력” 요약
- 강사별 현재 보유 수치 한눈에 표시

**일괄 반납**
- “전량 반납”을 한 강사에 대해 여러 교구 동시 선택 후 일괄 처리

---

## 8. 강사 관리 (`/admin/users`)

### 8.1 현황
- 강사 목록, 검색, 프로필 수정, 마일리지·수업 수 확인

### 8.2 추천 기능

**강사 대시보드**
- 강사 카드 클릭 시 “최근 수업·마일리지·재고” 요약 패널
- 수업 관리·마일리지 페이지로 바로가기 링크

**휴가 관리**
- `users.vacation` 편집 UI를 강사 상세에 통합
- 캘린더 형태로 휴가 기간 시각화(선택)

---

## 9. 마일리지 (`/admin/mileage`)

### 9.1 현황
- 강사별 points, 지급/차감, penalty, 로그

### 9.2 추천 기능

**기간별 조회**
- 월·분기 선택 후 해당 기간 지급/차감 합계
- 강사별 추이 차트(선택)

**내보내기**
- 기간·강사 선택 후 마일리지 로그 CSV/Excel

## 10. IIWarmup

### 10.1 Overview
- 각 스튜디오·AssetHub·Scheduler 진입 링크

**추천**: 최근 편집한 주차/월 메모(로컬 스토리지)로 “이어서 편집” 링크 제공

### 10.2 Play/Think Studio
- 편집·미리보기·시드 변경

**추천**: 프리셋 저장(로컬 또는 DB)으로 자주 쓰는 구성을 불러오기

### 10.3 AssetHub
- 이미지·BGM 업로드, 탭별 관리

**추천**: 드래그로 슬롯 순서 변경, “이전 주차에서 복사”로 에셋 일괄 복사

### 10.4 Scheduler
- 주차별 프로그램 배정, Publish

**추천**: “이전 달에서 복사”로 배정 패턴 재사용, 미리보기 링크 복사 버튼
