# SPOMOVE Movement Physical Contract

상태: **확정안 (O2 구현 기준)** — 승인 기록란은 운영자가 채움  
SSOT: [`SPOMOVE_OPERATION_LAYER_SSOT.md`](./SPOMOVE_OPERATION_LAYER_SSOT.md)

## 범위

**포함:** 목표 색 접촉 · 발·손 착지/지지 · 동작 완료 · 매트 위/밖 사용 · 필요 주변 공간 · 안전 · minimumCue / impact / jumpFree  
**제외:** 원위치 **복귀** (Operation: `startZone` + `timing`)

매트 가정: 스포매트 **60×60cm**, 색 칸 **약 30×30cm** × 4.

---

## 공통 금지

- 목표 색 **한 칸(30×30) 안에 양발 완전 착지**를 요구하지 않는다.
- 교실 전방 질주형 장거리 착지를 요구하지 않는다.
- Contract에 복귀·왕복을 적지 않는다.

---

## footTap (기존)

- 목표 자극 반응: 제시된 색 칸을 발(자유/같은쪽/교차)로 짧게 터치
- 시작 자세: Operation이 결정
- 발의 접촉 위치: 목표 색 칸 면
- 착지 방식: 해당 없음(터치)
- 동작 완료 기준: `briefContact`
- 매트 위/밖 사용 범위: 접촉은 매트 위
- 최소 필요 주변 공간: 매트 면
- minimumCueSeconds: 2 · impactLevel: low · jumpFree: true
- 쉬운 변형: 아래쪽 색부터 · 안전: 미끄럼 확인
- 금지 수행: 점프 착지 강요

## twoLegJump

- 목표 자극 반응: **양발이 동시에** 짧게 바닥을 떠나며 목표 색 **방향**으로 이동한 뒤, 발볼·발끝 등으로 목표 색 칸을 **식별 접촉**
- 시작 자세: 양발 평행, 무릎 살짝
- 발의 접촉 위치: 목표 색 칸의 일부(가장자리 포함). **양발 전체가 한 칸 안에 들어갈 필요 없음**
- 착지 방식: 양발 동시 착지(매트 위 또는 칸 경계 허용)
- 동작 완료 기준: `briefContact` — 홉 직후 목표 색에 발이 닿음
- 매트 위/밖 사용 범위: 주로 매트 위; 필요 시 매트 가장자리
- 최소 필요 주변 공간: 매트 면 + 사방 약 30cm
- minimumCueSeconds: 3 · impactLevel: high · jumpFree: false
- 쉬운 변형: 제자리 양발 Hop 후 손·발끝으로 색만 터치
- 안전 제한: 무릎·발목 부상 병력 시 제외 · 착지 소음·미끄럼 주의
- 금지 수행: 한 칸 양발 완전 수용 착지 · 높이 경쟁

권장 Operation 예시: `onMat + continuous`

---

## singleLegHop

- 목표 자극 반응: **한 발**로 목표 색 방향으로 홉하여 같은 발(또는 지정 발)로 목표 색을 접촉
- 시작 자세: 지지발 중심, 반대발 가볍게
- 발의 접촉 위치: 목표 색 칸
- 착지 방식: 홉한 발 착지·접촉 (완전 한발 정지 유지 불필요)
- 동작 완료 기준: `briefContact`
- 매트 위/밖 사용 범위: 매트 위
- 최소 필요 주변 공간: 매트 면 + 사방 약 30cm
- minimumCueSeconds: 3 · impactLevel: medium · jumpFree: false
- 쉬운 변형: 반대발 발끝 터치 허용
- 안전 제한: 균형이 무너지면 양발 착지 허용 후 재시도
- 금지 수행: 장거리 한발 도약

권장 Operation 예시: `onMat + responseWindow`

## boundingStep

- 목표 자극 반응: **매트 바로 밖(또는 인접)**에서 목표 색으로 **리드 발**이 칸을 접촉하는 힘찬 한 걸음(스키프/갤럽 리듬의 축소형)
- 시작 자세: 매트 밖 인접, 정면
- 발의 접촉 위치: 목표 색 칸 (리드 발)
- 착지 방식: 리드 발 접촉 후 트레일 발은 매트 밖 또는 가장자리 가능
- 동작 완료 기준: `briefContact`
- 매트 위/밖 사용 범위: **접근은 매트 밖, 접촉은 매트 위**
- 최소 필요 주변 공간: 매트 후방/측방 약 60cm
- minimumCueSeconds: 3 · impactLevel: medium · jumpFree: false
- 쉬운 변형: 보통 보폭의 큰 스텝으로 대체
- 안전 제한: 충돌 공간 확보
- 금지 수행: 교실 횡단 질주 · 매트만으로 장거리 바운드 정의

권장 Operation 예시: `adjacentToMat + responseWindow`

## plankTouch

- 목표 자극 반응: 플랭크(또는 무릎 플랭크)에서 **손**으로 목표 색을 터치
- 시작 자세: 하이플랭크 또는 무릎 플랭크, 발/무릎은 매트 밖 또는 가장자리 지지
- 발의 접촉 위치: 지지용(반응 아님). 손은 목표 색
- 착지 방식: 손 접촉
- 동작 완료 기준: `briefContact`
- 매트 위/밖 사용 범위: 손=매트 위, 하체 지지=매트 밖 허용
- 최소 필요 주변 공간: 매트 후방 약 80cm(다리 길이)
- minimumCueSeconds: 4 · impactLevel: medium · jumpFree: true
- 쉬운 변형: 무릎 플랭크 · 손목 부담 시 주먹/팔뚝 지지
- 안전 제한: 요통·손목 통증 시 제외 또는 무릎 플랭크만
- 금지 수행: 발만으로 색 반응

권장 Operation 예시: `adjacentToMat + responseWindow`

## quickStep

- 목표 자극 반응: 빠른 스텝으로 목표 색 **방향**을 맞춘 뒤, **발 1회** 목표 색 접촉 (연속 스텝 자체는 준비 리듬)
- 시작 자세: 양발 가벼운 바운스
- 발의 접촉 위치: 목표 색 칸 1회 명확 접촉
- 착지 방식: 스텝 접촉
- 동작 완료 기준: `briefContact` — 해당 큐에서 목표 색 1회 접촉
- 매트 위/밖 사용 범위: 매트 위
- 최소 필요 주변 공간: 매트 면
- minimumCueSeconds: 2 · impactLevel: medium · jumpFree: true
- 쉬운 변형: 제자리 빠른 스텝 후 천천히 터치
- 안전 제한: 과호흡·미끄럼 주의
- 금지 수행: “인터벌”을 움직임 정의에 포함 (인터벌은 Operation.timing)

권장 Operation 예시: `onMat + interval`
